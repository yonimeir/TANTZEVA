בחר משנה לאות

הוראות להפעלת האתר מקומית:

  1. התקן Node.js מ-https://nodejs.org/ אם טרם התקנת
  2. התקן את כלי ה-live-server הגלובלי: npm install -g live-server
  3. הפעל את השרת בתיקיית הפרויקט: live-server
  4. השרת יפתח אוטומטית את הדפדפן עם הכתובת http://localhost:8080

הערה: השימוש בשרת פיתוח מקומי פותר את בעיית ה-CORS בטעינת קבצי JSON מקומיים.
