# משניות לעילוי נשמת

מערכת אינטרנטית לבחירה וארגון של משניות ופרקי תהילים לעילוי נשמת נפטרים.
המערכת מאפשרת בחירת משניות לפי אותיות השם של הנפטר, ויצירת דף מותאם אישית להדפסה.

## תכונות עיקריות

- **דף משניות רגיל**: לימוד משניות לפי סדר, מסכת, פרק ומשנה
- **דף לעילוי נשמת**: הזנת שם נפטר ובחירת משניות המתאימות לאותיות השם
- **דף תהילים**: מציג פסוקי תהילים מפרק קי"ט ופרקים נוספים לפי אותיות השם
- **דף הדפסה**: יצירת דף לימוד מותאם אישית להדפסה

## טכנולוגיות

- HTML5, CSS3 ו-JavaScript טהור (ללא ספריות חיצוניות)
- שימוש ב-Sefaria API לטעינת טקסט המשניות וביאור הברטנורא
- שמירת נתונים ב-Local Storage
- תמיכה במכשירים ניידים ובדפדפנים מודרניים

## מבנה הקבצים

- **index.html**: עמוד הבית
- **mishnah_regular.html / mishnah_regular.js**: דף לימוד משניות רגיל
- **niftar.html / niftar.js**: דף בחירת משניות לעילוי נשמת
- **tehillim.html / tehillim.js**: דף בחירת פסוקי תהילים
- **print.html / print.js**: דף הדפסה
- **style.css**: עיצוב כללי
- **print.css**: עיצוב מיוחד לדף ההדפסה

### קבצי נתונים

- **mishnah_mappings.js**: מיפויים בין קודי מסכתות לשמות תצוגה ושמות API
- **mishnah_index.js**: אינדקס של כל המשניות בש"ס עם האות הראשונה של כל משנה
- **tehillim119_data.js**: טקסט מלא של תהילים פרק קי"ט מסודר לפי אותיות

## שימוש

1. **דף הבית**: בחר באחד מהדפים: לימוד משניות רגיל, לעילוי נשמת, תהילים או הדפסה

2. **דף משניות לע"נ**:
   - הזן את שם הנפטר
   - המערכת תחלץ את אותיות השם ותציג אותן בריבועים נפרדים
   - אפשר לבחור משניות באופן אקראי או ידני לכל אות
   - אפשר להוסיף את אותיות המילה "נשמה" ולבחור משניות ממקוואות

3. **דף תהילים**:
   - הזן את שם הנפטר
   - המערכת תציג את פסוקי תהילים קי"ט המתאימים לאותיות השם
   - אפשר לבחור פסוקים ספציפיים להוספה לדף הלימוד
   - אפשר לעיין בפרקי תהילים נוספים ובתפילות לאמירה בבית הקברות

4. **דף הדפסה**:
   - טעינת המשניות והפסוקים שנבחרו בדפים הקודמים
   - אפשרות לבחור פורמט: A4 רגיל או חוברת
   - אפשרות להכליל/להסתיר את פירוש הברטנורא
   - הדפסת הדף באמצעות מנגנון ההדפסה של הדפדפן
