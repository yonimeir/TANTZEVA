# אפליקציית משניות לעילוי נשמה - גרסה חדשה מסודרת

## תיאור
אפליקציה לבחירת משניות לעילוי נשמת נפטרים על פי אותיות השם. המערכת בנויה בארכיטקטורה מודולרית ונקייה עם הפרדה ברורה בין הרכיבים השונים.

## מבנה המערכת

### מודולים עיקריים

#### 1. MishnahCore.js - מודול הליבה
- **תפקיד**: מכיל את הלוגיקה המרכזית לשליפת משניות לפי אות
- **פונקציות עיקריות**:
  - `findMishnayotByLetter()` - שליפת משניות לפי אות (הלוגיקה המרכזית)
  - `getUniqueLettersFromNameOrdered()` - עיבוד שם נפטר לאותיות ייחודיות
  - `selectRandomMishnah()` - בחירה אקראית ממערך משניות
  - `formatMishnahForDisplay()` - פורמט משנה לתצוגה

#### 2. DataManager.js - מודול ניהול נתונים
- **תפקיד**: טעינה ואימות של קבצי הנתונים
- **פונקציות עיקריות**:
  - `loadAllData()` - טעינת כל קבצי הנתונים
  - `validateData()` - אימות תקינות הנתונים
  - `checkDataAvailability()` - בדיקת זמינות הקבצים

#### 3. StateManager.js - מודול ניהול מצב
- **תפקיד**: ניהול המשניות הנבחרות ושמירה ב-localStorage
- **פונקציות עיקריות**:
  - `addMishnah()` / `removeMishnah()` - הוספה והסרה של משניות
  - `saveToStorage()` / `loadFromStorage()` - שמירה וטעינה מהזיכרון המקומי
  - `exportData()` / `importData()` - יצוא ויבוא נתונים

#### 4. UIManager.js - מודול ממשק משתמש
- **תפקיד**: ניהול התצוגה והודעות למשתמש
- **פונקציות עיקריות**:
  - `updateUI()` - עדכון כללי של הממשק
  - `showError()` / `showSuccess()` - הצגת הודעות
  - `showMishnahSelectionDialog()` - דיאלוג בחירת משנה

#### 5. MishnayotApp.js - האפליקציה הראשית
- **תפקיד**: מחברת את כל המודולים ומנהלת את הלוגיקה הכללית
- **פונקציות עיקריות**:
  - `initialize()` - אתחול המערכת
  - `searchByLetter()` - חיפוש משניות לפי אות
  - `selectMishnah()` - בחירת משנה ספציפית

## קבצי נתונים

### קבצים חיוניים
- **mishnah_index.js** - אינדקס כל המשניות עם האות הראשונה
- **mishnah_mappings.js** - מיפויי שמות המסכתות (תצוגה ו-API)
- **mishnah_structure.js** - מבנה המשניות בכל פרק

### מבנה האינדקס
כל רשומה באינדקס מכילה:

```javascript
[masechetId, perekNum, mishnahNum, firstLetter]
// דוגמה: [1, 1, 1, "מ"] = ברכות א:א מתחילה באות מ
```

## הלוגיקה המרכזית

### שליפת משנה לפי אות
```javascript
function findMishnayotByLetter(targetLetter) {
    const candidates = [];
    
    // מעבר על כל האינדקס
    for (const entry of mishnahIndex) {
        if (Array.isArray(entry) && entry.length === 4) {
            const [masechetId, perekNum, mishnahNum, firstLetter] = entry;
            
            // בדיקה אם האות תואמת
            if (firstLetter === targetLetter) {
                candidates.push({
                    masechetId, perekNum, mishnahNum, firstLetter,
                    displayName: masechetIdToDisplayName[masechetId],
                    apiName: masechetIdToApiName[masechetId]
                });
            }
        }
    }
    
    return candidates;
}
```
